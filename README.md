# Bunyeth Portfolio

Personal portfolio website built with Vue 3 and Vite.

## Tech Stack

- Vue 3
- Vue Router
- Vite

## Project Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```

## Project Structure

- `src/components` shared UI components
- `src/views` route pages
- `src/composables` reusable logic
- `src/router` app routes
- `src/assets` global styles
